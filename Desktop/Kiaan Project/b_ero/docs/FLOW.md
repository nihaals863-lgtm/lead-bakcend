# System Flow

## Lead Flow

1. Lead created (Admin / System)
2. Status = new

3. Assigned to Manager
4. Manager assigns to Agent

5. Agent updates status:
   new → contacted → qualified

6. Agent converts lead:
   qualified → converted

7. System creates Contact

---

## Rules

- Cannot skip status
- Cannot convert without qualified
- Converted lead is locked

---

## Contact Flow

- Created only from Lead
- Editable anytime

---

## Deal Flow

1. Deal created from Contact
2. Assigned to Agent
3. Status updated

---

## Commission Flow

- Generated after deal closed
